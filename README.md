# Periwinkle UI

## Installation

To use Periwinkle UI components, all you need to do is install the `periwinkle-ui` package

```sh
# npm
npm i periwinkle-ui

# yarn
yarn add periwinkle-ui

# pnpm
pnpm add periwinkle-ui
```

```jsx
import { Button } from 'periwinkle-ui'

function App() {
  return <Button />
}
```
